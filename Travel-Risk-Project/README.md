# RouteWise

RouteWise builds on my traffic analytics project by combining live traffic conditions with real-time weather data to estimate overall travel conditions across New York City.

Rather than looking at congestion alone, RouteWise considers multiple factors that can affect a commute and summarizes them into a single travel recommendation.

## What it does

- Combines live traffic and weather data into one analysis
- Calculates an overall travel risk score
- Evaluates rain probability, wind gusts, and visibility
- Includes rush-hour conditions in the final score
- Provides an easy-to-understand travel recommendation
- Visualizes travel conditions on an interactive map

## Technologies

- Python
- pandas
- requests
- Folium
- TomTom Traffic API
- Open-Meteo API
- Jupyter Notebook
